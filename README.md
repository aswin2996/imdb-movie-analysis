# 🎬 IMDB Movie Data Analysis

This project analyzes the IMDB Top 1000 movies dataset using Python.

## 📊 Tools Used:
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 🔍 Analysis Performed:
- Data Cleaning
- Histogram Visualization
- Box Plot for Outliers
- Correlation Heatmap
- Hypothesis Testing (T-test)
- Linear Regression

## 📈 Key Insight:
IMDB ratings show a weak correlation with movie revenue, indicating that higher ratings do not guarantee financial success.
